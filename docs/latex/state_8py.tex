\doxysubsection{Detailed Description}
Problem state representation for the Beluga Challenge. 

This module contains all classes and functions for representing and manipulating the state of the Beluga Challenge container optimization problem. It includes data structures for jigs, belugas, racks, production lines, and the complete problem state with all necessary operations for MCTS and RL algorithms.

Classes\+:
\begin{DoxyItemize}
\item Jig\+Type\+: Represents different types of jigs with size properties
\item Jig\+: Individual jig instance with type and empty/loaded status
\item Beluga\+: Container ship with current and outgoing jigs
\item Rack\+: Storage rack with size constraints and current contents
\item Production\+Line\+: Factory production line with scheduled jigs
\item Problem\+State\+: Complete state representation with all components 
\end{DoxyItemize}
