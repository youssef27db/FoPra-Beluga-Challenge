13class SkdSPDDLDomain(SkdBaseDomain, RLDomain, FullyObservable):
14 """Reinforcement learning scikit-decide domain, based on a encoding of the actions to model the logics of
15 the transition function. The methods of this class should not be used directly, but rather through the
16 public methods of the domain feature class from which this class derives:
17 https://airbus.github.io/scikit-decide/reference/_skdecide.domains.html#rldomain
18
19 Important note #1: in addition to standard gymnasium-like domains, this class also provides access to the
20 set of applicable actions in a given state with the `get_applicable_actions(memory: State)` method via the
21 inherited RLDomain compound domain feature class. The `step(action: Action)` method will raise an exception
22 if the passed action is not applicable in the current state, i.e. in the state returned by the previous call
23 to the `step(action: Action)` method or to the `reset()` method. Therefore, it is important to get the set
24 of applicable actions with the `get_applicable_actions(memory: State)` method in the current state before
25 calling the `step(action: Action)` method.
26
27 Important note #2: the simulator is based on a random shuffle of the ordering of Beluga flights when calling
28 the `reset()` method at the beginning of each episode, from which an episode-specific PDDL file is generated
29 to define the logics of the transition function in the `step(action: Action)` method. It means that different
30 calls of the `step(action: Action)` from the same internal PDDL state but within different episodes will result
31 in different next states (since the PDDL problems in each episode are different). In order to avoid misuse and
32 misinterpretation of the internal state by the solver, we intentionally hide the internal state of the domain,
33 which is also consistent with gym-like definitions of reinforcement learning domains.
34
35 Args:
36 SkdBaseDomain (_type_): Base domain
37 RLDomain (_type_): Compound scikit-decide class importing reinforcement learning domain features
38 FullyObservable (_type_): Domain feature class making the RL domain fully observable
39 """
